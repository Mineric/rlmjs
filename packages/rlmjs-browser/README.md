# @software-machines/rlmjs-browser

Browser runtime package for `rlmjs`.

Provided:
- `IndexedDbStorageAdapter` for browser-static context storage.
- `createWorkerToolRuntime()` and `bindWorkerToolHandler()` for worker isolation.
- `HttpStorageAdapter` for optional backend RPC storage.

Usage example:
- `experiments/rlmjs-browser-static-demo/README.md`
